關於我們

SITCON 學生計算機年會，由對資訊抱持熱誠的學生們所組成的同名社群自發籌辦，希望能透過技術與知識的激盪，給予學生們一個用自身力量實踐夢想的舞台。本屆年會之主軸聚焦於真正的駭客精神，The True Hackers 。希望讓大家瞭解駭客精神的中心思想，以及駭客精神在各個領域的展現。

The True Hackers

駭客 (hacker) 起源於 1960 年代的麻省理工學院 (MIT) 人工智慧實驗室 (CSAIL) ，在電腦及計算機科學剛起步的年代，初代駭客們想要了解電腦每一部分、每一位元的運作，就是為了寫出更精簡但卻更強大的程式碼，在硬體資源極為有限的狀況下，仍得以探索電腦的用途，一暼運算的未來。

他們說 ”Information wants to be free.” ，資訊不應由少數人壟斷，他們早已看透，電腦普及之後，資訊的傳播複製將會以數千倍數萬倍的速度且幾乎零成本的方式進行。有了電腦，更應該讓資訊自由地流通，而非限制它。他們更致力於將電腦普及化，他們相信電腦可以改善人的生活，電腦不應被鎖在企業與政府的層層權威與官僚當中。

初代駭客們的理念影響了無數人，在各個領域當中發芽茁壯，無論是自由軟體運動、資訊安全、自造者運動、和吹哨者們，都是駭客精神的展現。

議程種類

Talk / Short Talk

Talk 為 30 分鐘的議程，加上10分鐘的問答時間； Short Talk 為 10 分鐘的議程，問答時間由講者自行安排。兩者僅有時間上的差別。

Lightning Talk

大會當天早上開放報名，每位講者有3分鐘，3分鐘一到可是會被趕下臺的！

投稿方式

稿件均經議程組形式審查，交予該領域之審稿委員匿名審稿；審稿委員由業界人士、學界人士及 SITCON 議程組員組成，依照下列評分方式評分，錄取高分者，稿件一律以速審為原則。

開始投稿

期限

  • 2015/1/31（六）- 6:38 AM 投稿截止
  • 2015/2月上旬 - 審稿結束
  • 2015/2月中旬 - 公告完整議程

格式（Talk 與 Short Talk 的投稿格式相同）

  • 題目：必填，15 全形字以下
  • 投稿類型：請寫明所欲投稿者為 Talk 或 Short Talk。
  • 暱稱：公開於講者海報及網站，如未填寫，將使用本名。
  • 個人介紹：必填，含標點符號 50 ~ 100 全形字，安排於講者海報及網站。
  • 個人頭像：選填，請提供解析度 1000x1000 以上之 JPG/PNG 圖檔或連結。
  • 摘要：必填，含標點符號 150 ~ 250 全形字，安排於講者海報及網站。
  • 詳細說明：不限字數，可斟酌繳交，或附上程式碼、投影片草稿等，以利審稿委員更瞭解你的作品。
  • 個人聯絡資訊：必填，含真實姓名、手機號碼、電子信箱、所屬學校或單位、職稱或系級，僅供大會聯絡及印製感謝狀使用。
  • 出發地：平時生活的地區，必填，僅供大會調查統計使用。

注意事項

  • SITCON 2015 的舞臺設計將大不同於以往，請參考舞臺設備、錄影及直播
  • 為增加入選機率，請參考我們的評分方式
  • 為鼓勵投稿，大會將提供投稿人一組 VIP Code （一人一組，非一稿一組），可優先報名參與本年會，獲選講者請使用講者身分入場，並可將 VIP Code 給他人使用。
  • 在截稿之前，投稿者可以隨時使用投稿系統修改已投的稿件
  • SITCON 議程組得經作者同意，轉換 Talk 或 Short Talk 稿件之投稿類型。
  • 無論有無入選，仍然歡迎於年會當天報名 Lightning Talk！
  • 凡稿件經接受者，年會將頒予感謝狀以表彰投稿人之熱情付出與貢獻。
  • 講者海報僅於年會當日於年會場地張貼。

SITCON 2015 採用投稿系統接收投稿，需先註冊帳號，之後登入新增投稿。（同一帳號僅能為一人使用，一人能投多份稿件）

一個半形字元計為0.5個全形字元，一切投稿相關規定事項以本文件為準。

若有任何問題請 email 至 SITCON 議程組信箱（session@sitcon.org）詢問。

評分方式

審稿委員主要針對以下項目進行評分

  • 稿件內容：包含的知識、經驗、想法、觀點等，還有是否有趣、酷炫
  • 稿件難易度：是否需要大量背景知識，我們偏好不需大量背景知識的稿件
  • 表達能力：提供的資料是否有條理、文句暢通，提供的資料（尤其是稿件的詳細說明）的完整度，我們偏好完整的資料，讓審稿委員更清楚了解演講細節

舞台設備

這次年會提供的舞台設備與往年不同，若講者事先提供投影片，投影片可以由中控台播出，講者使用簡報筆控制，可省去現場連接電腦的麻煩和風險。 講者仍然可以選擇使用自己的電腦播出投影片，現場因為高畫質直播的關係，會採用 HDMI 數位訊號。

講台上備有供講者簡報使用的 iMac（會搭配設定好的簡報筆），以及 2個 HDMI 外接訊號（含 Thunderbolt to HDMI 轉換器及 Displayport to HDMI 轉換器），如果講者因為 Live Demo 或是任何原因需使用自有設備，且該設備僅有 VGA 訊號，歡迎隨時聯絡議程組，若錄取後我們亦會協助您處理設備相容性問題。

錄影及直播

所有議程皆會錄影及線上直播，若不希望錄影可以與議程組聯絡。講者可以選擇將錄影以創用CC或其他授權方式釋出。

為了配合線上直播的需求，本屆 SITCON 提供入選講者試講及彩排的機會。試講除了讓大會工作人員先行對議程內容、演講習慣有初步了解外，也可以讓講者熟悉現場流程、找出能讓簡報更生動精采的模式。彩排則讓講者熟悉現場設備環境、攝影機位置，並且測試 Live Demo 情境。希望講者能夠共同參與試講及彩排，一起使年會直播品質更臻完美。
